# vue3 创建项目
+ npm init vite-app  projectName
+ cd projectName
+ 安装依赖： npm install 
+ 启动： npm run dev 
# 安装axios
+ npm install --save axios vue-axios
+ 注意：不在使用axios,不支持跨域
# 安装 ant-design-vue
+ npm i --save ant-design-vue@next
# 安装 scrypt-async
+ npm install scrypt-async
- 注意修改导出代码：export default scrypt
# 安装vuex
+ npm install vuex@next --save
# 安装vue-Router
+ npm install vue-router@next
